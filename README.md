# ZookeeperClient  
ZooKeeper客户端，支持NET46和NETSTANDARD2_0，实现了断线重连，会话过期重连，永久监听，子节点数据变化的监听。并且加入了常用功能，例如分布式锁，Leader选举，分布式队列。该项目被入选donet跨平台公众号 http://mp.weixin.qq.com/s/DD-DGp6f34JYSHDlat1TdA
